# Security Policy

## Supported Versions

Die aktuelle Release-Version ist 1.0.0, diese beinhaltet die Grundfunktionalität von Feedibus und wird unterstützt. Für alle Pre-Release Version gewährleisten wir keine Funktionsfähigkeit

| Version | Supported          |
| ------- | ------------------ |
| 1.0.0   | :white_check_mark: |
| 0.x.x   | :x:                |
